# 배달 주문 웹서비스 delivery

-----
## 개요

-----
인프런에서 김영한님의 JPA와 관련된 강의를 모두 듣고 평소에 자주 사용하던 배달 주문 서비스인 
배달의 민족과 요기요를 참고하여 배달 주문 서비스 delivery를 만들어 보았다.

* 사용 기술 및 개발 환경
  * H2 데이터베이스
  * Spring MVC, Spring Data JPA, QueryDSL
  * HTML, Bootstrap4, Thymeleaf
